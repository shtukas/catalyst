Dispatch items are json object with at least
    - uuid
    - mikuType

Items are in a database 
    CREATE TABLE items (uuid TEXT primary key, mikuType TEXT, item TEXT);
    See Items.rb for details

Cross types conventions (all optional):
    - skip-0843             : unixtime (skipped until this time)
    - do-not-show-until-51  : null or DateTime Iso8601 # 2025-11-02T21:49:59Z
    - payload-37            : null or UxPayload
    - donation-14           : null or String # bank account to donate to
    - engine-1437           : NxEngine or null
