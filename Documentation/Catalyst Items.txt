Catalyst items are json object with at least
    - uuid
    - mikuType

Database:
    create table Items (_uuid_ TEXT primary key, _mikuType_ TEXT, _item_ TEXT)

Cross types conventions (all optional):
    - skip-0843             : unixtime (skipped until this time)
    - do-not-show-until-51  : null or DateTime Iso8601 # 2025-11-02T21:49:59Z
    - nx42                  : null or Float # Positioning
    - payload-37            : null or UxPayload

    - donation-13           : null or Array[String]
                              bank accounts to donate to

    - active-67             : boolean
                              indicates that the item was selected
                              during morning, and should remain in
                              the context.

    - duration-38           : duration in minutes for the morning planning
