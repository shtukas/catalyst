Catalyst items are json object with at least
    - uuid
    - mikuType

Database:
    create table Items (_uuid_ TEXT primary key, _mikuType_ TEXT, _item_ TEXT)

Cross types conventions (all optional):
    - skip-0843             : unixtime (skipped until this time)
    - do-not-show-until-51  : null or DateTime Iso8601 # 2025-11-02T21:49:59Z
    - payload-37            : null or UxPayload
    - donation-14           : null or String # bank account to donate to
    - global-pos-07         : global positioning, Real
    - timecore-57           : TimeCore57
    - is-priority-02        : null or array of dates $1

TimeCore57 
    - uuid                  : String
    - name                  : String
    - day-expectation-hours : REAL or null defaults to 1

$1 if the current date is listed, then it's a priority item
